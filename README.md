# Network Diagnostic Tool

A Python-based command-line tool to diagnose basic network connectivity issues.

## Features
- DNS lookup (domain to IP)
- Ping host to check reachability
- Port scanning (HTTP/HTTPS)
- Response time measurement

## Tech Used
- Python (socket, subprocess)

## How to Run
1. Install Python
2. Run: python network_tool.py
3. Enter a domain (e.g., google.com)

## Purpose
This project was built to understand networking fundamentals like TCP/IP, DNS resolution, and basic troubleshooting techniques.
